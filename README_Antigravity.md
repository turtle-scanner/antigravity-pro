# ⚖️ 안티그래비티 프로 터미널 (v5.5 Ultimate) 인수인계 가이드

본 파일은 거북이투자전문가님의 안티그래비티 사령부를 성공적으로 운영하고, 추후 고도화 작업을 원활하게 이어가기 위한 **공식 매뉴얼**입니다.

## 📋 프로젝트 개요
- **목표**: 프라딥 본데(Bonde) 철학 기반의 13개 핵심 분석 도구 및 커뮤니티 통합 사령부 구축.
- **최종 빌드 날짜**: 2026-04-18
- **주요 엔진**: 전용 SQLite 데이터베이스 (`antigravity_master.db`) 기반 동시접속 대응 아키텍처.

## 🗂️ 주요 파일 구성
1. `full_terminal_app_v5.5_Integrated_Final.py`: 전체 시스템 소스 코드 (UI, 인증, 분석 엔진 통합).
2. `requirements.txt`: 배포 및 실행에 필요한 Python 라이브러리 명세서.
3. `antigravity_master.db`: 회원, 대화, 포트폴리오 데이터가 저장되는 실제 데이터베이스 파일.

## 🛠️ 핵심 기능 요약 (Resume Points)
- **1번 메뉴 (스캐너)**: BMS 점수 기능 탑재, 하이라이트 스타일링 적용.
- **2번 메뉴 (소통방)**: 100개 단위 페이징 시스템 및 구글 시트 실시간 동기화.
- **12번 메뉴 (리스크)**: 포트폴리오 영구 저장 기능 (Json에서 SQL로 업그레이드됨).
- **13번 메뉴 (히트맵)**: Plotly 기반 실시간 주도주 온도계.
- **배경 & 사운드**: 강제 다크 모드(CSS) 및 사이드바 ASMR 오디오 엔진 최적화.

## 🚀 배포 업데이트 프로세스 (GitHub)
나중에 코드를 수정하거나 업그레이드할 때 순서:
1. 로컬에서 코드 수정 후 테스트.
2. GitHub 저장소(`antigravity-pro`)에 파일 업로드 및 **Commit**.
3. Streamlit Cloud 대시보드에서 자동으로 빌드가 시작됨을 확인.

## 💡 전문가님을 위한 관리 팁
- **DB 백업**: `antigravity_master.db` 파일은 전문가님의 소중한 자산입니다. 한 달에 한 번 정도 로컬로 다운로드하여 백업해 두는 것을 권장합니다.
- **시트 관리**: `MASTER_GAS_URL` 주소는 전문가님의 구글 앱스 스크립트와 연결되어 있습니다. 시트 구조를 바꾸시면 스크립트 배포를 다시 해주어야 동기화가 유지됩니다.
